# Game for Ludum Dare 51

A game for the Ludum Dare 51 compo written in Zig for the WASM-4 virtual console platform.

## Building

Build the cart by running:

```shell
zig build -Drelease-small=true
```

Then run it with:

```shell
w4 run zig-out/lib/cart.wasm
```
